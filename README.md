# Simple To-Do List Manager

## Description
This is a simple Python-based To-Do List application that allows users to:

- Add new tasks
- View existing tasks
- Exit the program

The program uses a list to store tasks and a loop to keep the application running until the user chooses to exit.

## Features
- Add tasks dynamically
- Display all saved tasks
- User-friendly menu-driven interface
- Simple and beginner-friendly Python project

## Technologies Used
- Python 3

## How It Works

### Menu Options

1. **Add Task**
   - Allows the user to enter a task.
   - The task is stored in a list.

2. **View Tasks**
   - Displays all tasks currently stored in the list.

3. **Exit**
   - Terminates the program.
